# complex-systems-instruments
> Exploring how information-theoretic quantities behave in stigmergic ant colonies. This is a prototype built to investigate questions about collective systems, not evidence for causal emergence.

Stigmergy Information Instrument (v1)
> An interactive research instrument exploring information-theoretic measurements in a Deneubourg-style ant colony simulation.
>  This project investigates effective information, determinism, degeneracy, and phase transitions in collective systems. It is intended as a research prototype rather than a demonstration of a scientific result.
